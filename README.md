# 🎮 Pixel Bounce

A fast-paced HTML5 Canvas arcade game. Bounce your ball upward between platforms, collect golden stars, and climb as high as you can!

## 🕹️ Play Now

**[▶ Play Pixel Bounce](https://jperezdelreal.github.io/pixel-bounce/)**

## How to Play

- **Desktop:** Arrow keys or WASD to move left/right
- **Mobile:** Touch left/right side of screen to move
- **Goal:** Bounce higher, collect ⭐ stars, beat your high score
- **Game Over:** Don't fall off the bottom of the screen!

## Features

- 60fps smooth gameplay
- Touch + keyboard controls
- Moving platforms (green)
- Star collectibles (+25 points)
- High score persistence (localStorage)
- Particle effects on bounce and star collection
- Responsive, mobile-friendly

## Tech

- Pure HTML5 Canvas — zero dependencies
- Single `game.js` file, instant load
- GitHub Pages deployment
